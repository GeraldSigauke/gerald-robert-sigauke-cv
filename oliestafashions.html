Built The Frontend Portion For An Ecommerce Web App

  • Source Code
  • Demo

This React Js project is a eleven page web app which serves as the frontend portion for an ecommerce web app. The routing is handled by "React-Router-Dom". The Header is a three part navbar with a search field. The Home page makes use of the React-Fast-Marquee node package. Bootstrap 5 was used for the styling. This project is still a work in progress.

Here is a brief summary of all the bells and whistles:

  • I used "React-Router-Dom" for all the routing.
  • I used Bootstrap 5 for the styling.
  • I used the React-Fast-Marquee node package for the logo clouds.
