Built A Landing Page For An Afterschool Care Business

  • Source Code
  • Demo

This React Js project is a six part landing page for an afterschool care business. Everything is fully responsive. The Navbar has a bespoke toggle for switching between light mode and dark mode. The Hero section makes has two 'buttons'. One takes the user to the Contact section. The other takes the user to the About section. The Carousel and the Testimonials section make use of the Swiper node package. The Contact section has a map for directions. it also makes use of FormSubmit to send an email to the oliestaafterschoolcare@gmail.com. The footer section makes use of the React-Icons node package.

Here is a brief summary of all the bells and whistles:

  • I used tailwind CSS for the styling.
  • The Navbar has a bespoke toggle for switching between light mode and dark mode.
  • I used React Icons for all the icons.
  • The Carousel uses the Swiper node package.
  • I used FormSubmit for sending emails.
