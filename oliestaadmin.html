Built A Dashboard Or An Admin Section For A Web App

  • Source Code
  • Demo

This React Js project is a five page app which serves as the dashboard or an admin portion for a web app. The routing is handled by "React-Router-Dom". It can switch between light mode and dark mode. I used a combination of Material UI and SASS for most of the UI. For the circular progress bar, I used React Circular Progressbar. I used Recharts for the line graph. This project is still a work in progress.

Here is a brief summary of all the bells and whistles:

  • I used Material UI and SASS for the UI and styling.
  • It can switch between light mode and dark mode.
  • I used React Circular Progressbar for the circular progress bar.
  • I used Recharts for the line graph.
